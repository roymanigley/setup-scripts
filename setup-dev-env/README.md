# Initialising the DEV environment
> following tools will be installed:
- ohmyzsh 
- git 
- curl 
- wget 
- vim 
- build-essentials 
- meld
- docker
- sdkman
  - java
  - maven
- nvm
  - node
- JetBrains Toolbox

1. initial install:

	source init-ubuntu

2. installing the SDKs:  

	source init-sdks
